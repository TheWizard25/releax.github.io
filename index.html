Features

Independent

releax os is a independently developed linux distro build from scratch, means new adventure, new everything

Customizable

Obviously its GNU linux a open source linux distro with xfce 4.14 to provide ultimate level of customizablitiy,so that you os with look like in the way you want

Bootup Init

a simple and modern approach of traditional unix init scheme, homegrown by relax, uses traditional and simple /etc/init.d for service manage

Free

releax os is free to modify, distribute, redistribute, and whatever you want to do with it

Chatbot

releax os have a homegrown chatbot or assistant that help you to manage your linux, not like other chatbot, it can do a lot of automative task and a very easy programming

pkgs

a homegrown innovative package manager for gnu/linux that is capable of handling both source packages from recipies (BSD like ports) and binary packages (tar.xz) with extension .pkg
